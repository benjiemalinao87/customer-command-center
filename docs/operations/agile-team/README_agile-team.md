# Agile Team Playbook (Startup)

This folder contains the **team operating system** for building Automate8 (LiveChat CRM) with a small Agile team.

## What’s inside

- **HTML hub (presentations)**: [`index.html`](./index.html)
- **Playbook (start here)**: [`agile-team-operating-model.md`](./agile-team-operating-model.md)
- **Roles & decision-making**: [`roles-and-alignment.md`](./roles-and-alignment.md)
- **Meetings (scripts + agendas)**: [`meeting-templates.md`](./meeting-templates.md)
- **User stories system (templates + DoR/DoD)**: [`user-story-system.md`](./user-story-system.md)
- **Code push rules**: [`code-push-rules.md`](./code-push-rules.md)
- **Interview process**: [`interview-process.md`](./interview-process.md)
- **Presentation (HTML deck)**: [`presentation-team-agile.html`](./presentation-team-agile.html)

## How to use this (Sprint 0)

1. Read the playbook (15 minutes).
2. Run the **first meeting** using [`meeting-templates.md`](./meeting-templates.md).
3. Create your first **10–20 user stories** using [`user-story-system.md`](./user-story-system.md).
4. Start Sprint 1 and demo every week.
